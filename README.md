# Map Service

Микросервис карты для мобильного приложения с использованием Яндекс.Карты API.

## Описание

Сервис позволяет пользователям воспользоваться функциональном Яндекс.Карт в приложении.
Пользователь может ценрировать карту на своём местоположении, узнать расстояние до ближайших предприятий и выбрать желаемое предприятие.

## Технологии

- FastAPI
- Python 3.11+
- Яндекс Карты API
- Docker
